# 📧 Spam Email Classifier AI

This project was developed as part of the KODBUD AI Internship.

## Description

This Machine Learning project classifies messages as Spam or Not Spam.

## Technologies Used

* Python
* Google Colab
* Scikit-learn
* TF-IDF
* Naive Bayes

## Features

* Text preprocessing
* Spam detection
* AI-based prediction

## 🚀 How to Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. Enter a message.
4. View the Spam/Not Spam prediction.
